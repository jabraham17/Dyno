# Dyno

A very basic AST manipulation library for Chapel. This serves as a proof of concept to interop with the Chapel compiler itself to manipulate the AST. The API is not fleshed out yet, please open an issue with any features you'd like to see or open a PR if you'd like to contribute!

## Installation

Add Dyno as a Mason dependency:

```bash
mason add Dyno@0.2.0
```
